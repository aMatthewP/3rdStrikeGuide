So you want to play 3rd Strike?

The first thing you need to do in order to setup 3rd strike on fightcade is to first download fightcade of course. After going to the site download the version which matches your OS.
❮

❯
After you have successfully downloaded fightcade you will need to download the json files from this link. After downloading the json files open the zip file you have downloaded and copy the json files inside the zip file and paste them into the following directory Fightcade\emulator.
❮

❯
Open fightcade and click on the search icon and search for 3rd strike choose the one with the most players then click the join button. A terminal should appear which indicates that it is downloading the correct roms for 3rd strike. On the left you should see the 3rd strike lobby click on it to go to it (Note: you can only join 3 lobbies at once). After joining the lobby click on the test game button on the top right corner to play offline. A window should pop up that is the emulator the will run the game. Click on the input button on the top and then choose map inputs or alternatively press F5. Then map the buttons with the inputs that you want.
❮

❯
Now that you have played abit of 3rd strike offline you might want to try to git gud. On the top right of the fightcade menu next to the test game option you will see a training button which will start a training version of the game. You will have to map player to controllers to select the player 2 in training mode. Alternatively you can download the following lua files from this github repository click on code and download zip. After downloading the zip file create a folder in the fightcade\emulator\fbneo directory called lua and paste the contents of the zip file into the folder you created. Then open up fightcade start up 3rd strike. Once 3rd strike is opened click Game and New lua scripting then new lua script window then a window should pop up click browse then find and open the lua folder that you created and double click the file called 3rd_training_lua and then click Run. For details on how to use this training mode refer to this link for more information.
❮

❯
Now that you can play 3rd strike offline and have trained you now might want to try playing online and get bodied by good players lol. To play online on the looking to play sectionright click on a player that you want to play with and click challenge. The bars on the right indicate your connection if you were to play with that person, hovering over it show you their ping.
